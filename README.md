Лабораторная работа №16
Создание приложения с графическим интерфейсом.
Использование awt.
В данной программе реализовано оконное приложение. В первом диалоговом окне есть кнопка, при нажатии на которую
открывается второе диалоговое окно с двумя кнопками. Первая завершает работу всей программы, вторая закрывает только второе диалоговое окно 
и пользователь возвращается к первому.
